VPS搭建自己的ShadowSocks

https://github.com/shadowsocks

Windows ShadowSocks使用说明

  • 说明地址：https://github.com/shadowsocks/shadowsocks-windows
  • 软件下载地址：https://github.com/shadowsocks/shadowsocks-windows/releases，版本：Shadowsocks-4.0.6
  • 需要安装 .NET Framework 4.6.2 https://www.microsoft.com/zh-CN/download/details.aspx?id=53344
  • 在任务栏找到 Shadowsocks 图标；在 “服务器” 菜单添“编辑服务器”
    • 服务器地址：139.162.42.77
    • 服务器端口：8381～8385，任选一个
    • 密码：8381～8385端口对应foobar1～foobar5
    • 加密：aes-256-cfb
  • 选择 “启用系统代理”来启用系统代理。请禁用浏览器里的代理插件，或把它们设置为使用系统代理

Mac OS ShadowSocks使用说明

  • 说明地址：https://github.com/shadowsocks/ShadowsocksX-NG
  • 软件下载地址：https://github.com/shadowsocks/ShadowsocksX-NG/releases/；要选择对应系统版本，不是最新系统，选择V1.4是可以使用的
  • 在任务栏找到 Shadowsocks 图标；在 “服务器-新服务器” 菜单添“服务器设置”
    • 地址：139.162.42.77：8381～8385
    • 加密方法：aes-256-cfb
    • 密码：8381～8385端口对应foobar1～foobar5

Ubuntu ShadowSocks使用说明

  • 说明地址：https://github.com/shadowsocks/shadowsocks-qt5/wiki
  • 软件安装方法：PPA is for Ubuntu >= 14.04.
    • sudo add-apt-repository ppa:hzwhuang/ss-qt5
    • sudo apt-get update
    • sudo apt-get install shadowsocks-qt5
  • 在任务栏找到 Shadowsocks 图标并启动
  • 浏览器插件：
    • Firefox前往 https://addons.mozilla.org/zh-CN/firefox/addon/foxyproxy-standard/ 安装 foxyproxy-standard 扩展
    • 单击新建代理服务器 Add New Proxy
      Manual Proxy Configuration
      Host or Ip Address地址为 127.0.0.1
      Port端口为 1080
