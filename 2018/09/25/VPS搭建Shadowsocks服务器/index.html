​ shadowsocks 是一款自定义协议的代理软件，由于其流量特征不明显，（直到不久前）一直可以稳定提供上网代理。shadowsocks 客户端会在本地开启一个 socks5 代理，通过此代理的网络访问请求由客户端发送至服务端，服务端发出请求，收到响应数据后再发回客户端。

    • 这里经过对比，选择了Linode

  • 远程登陆 VPS

    • Mac OS X 或 Linux 下直接在终端中 ssh root@your_vps_ip -p your_ssh_port 即可

  • 安装 shadowsocks（Ubuntu/Debian系统）

  • 编写配置文件

    shadowsocks 启动时的参数，如服务器端口，代理端口，登录密码等，可以通过启动时的命令行参数来设定，也可以通过 json 格式的配置文件设定。推荐使用配置文件，方便查看和修改。

配置文件中个字段的含义：

    • server: 服务器ip地址

    • possword: 密码
    • timeout: 超时时间
    • method: 加密方法
    • fast_open: 如果你的服务器 Linux 内核在3.7+，可以开启 fast_open 以降低延迟
    • workers: 默认为1
