记录Debian7.7搭建PyQt5环境

Oct 13, 2014

debian7.7 环境下搭建Python3.2 + PyQt5.3.2 + Eric6.0

注：此debian7.7是（stable）稳定版本，在aptitude中无pyqt5相关的包文件，故采用以下安装。debian（testing）测试版本的aptitude中有pyqt5的包，因此可以直接安装，无须以下步骤。

  • 安装Python3

​ 在debian7.7默认安装的世python2.7，使用Python3可以直接用aptitude install python3 (默认安装python3.2.3)。在终端使用Python3命令即可。

  • 安装Qt5

    由于debian stable版本中有自带qt4的部分库文件，但我们需要qt5的开发平台，所以首先安装qt5。

Qt5.3.2下载：http://qt-project.org/downloads 。

安装过程：

​ 下载到的文件可以是在线安装程序，也可以是完整的包。在线：qt-opensource-linux-x86-1.6.0-5-online.run 完整：qt-opensource-linux-x86-5.3.2.run

  2. 注，一般su安装默认在/opt/下，普通用户安装在/home/jinfei(用户名)/下。这里我采用是完整包安装。

配置过程：

​ 安装完成后配置一下PATH，在当前用户主目录下的.bashrc （普通用户～/.bashrc，su用户/root/.bashrc）文件中加入：

在终端输入qmake，如果显示qmake命令不存在，请通过创建软连接进行解决：将/usr/bin下的qmake软连接到/home/jinfei/Qt5.3.2/5.3/gcc/bin/qmake上，或者将/usr/lib/i386-linux-gnu/qt4/bin/qmake软连接到/home/jinfei/Qt5.3.2/5.3/gcc/bin/qmake上：

此时再输入qmake查看是否设置成功。使用qmake -v查看。

  • 安装sip

    ​ SIP是python调用C/C++库的必备模块。因 此SIP是PyQt的依赖工具，安装PyQt之前必须先安装对应版本的SIP（PyQt5要求sip必须是sip4.16之后的）。PyQt编译时使用的SIP版本必须与python默认调用的SIP保持一 致！否则python中是无法调用PyQt的。这里我们使用最新的SIP 4.16.4.终端输入sip -V即可查看当前版本。

  1. ​ SIP 4.16.4下载：http://www.riverbankcomputing.com/software/sip/download

  2. 安装过程：

  3. tar -zxvf PyQt-gpl-5.3.2.tar.gz
    cd PyQt-gpl-5.3.2
    python3 configure.py

    Aptitude install libgl1-mesa-dev libglu1-mesa-dev
    再次执行python3 configure.py 。

sudo make
sudo make install

sudo ln -s /usr/lib/python3.2/site-packages/PyQt5 /usr/lib/python3.2/PyQt5

sudo ln -s /usr/lib/python3.2/site-packages/PyQt5 /usr/lib/python3/dist-packages/PyQt5

python3 configure.py –destdir /usr/lib/python3.2

tar -xzvf QScintilla-gpl-2.8.4.tar.gz

cd QScintilla-gpl-2.8.4

cd Qt4Qt5
qmake qscintilla.pro
sudo make
make install

cd ../designer-Qt4Qt5

qmake designer.pro

sudo make

sudo make install

cd ../Python

python3 configure.py –pyqt=PyQt5

sudo make

sudo make install

tar -zxvf eric6-6.0-snapshot-20140720.tar.gz
tar -zxvf eric6-i18n-zh_CN.GB2312-6.0-snapshot-20140720.tar.gz //语言包文件会自动解压到Eric6.0主文件解压目录中。没有的话手动复制。
cd eric6-6.0-snapshot-20140720
sudo python3 install.py //安装主程序
sudo python3 install-i18n.py //安装中文语言包

sudo chmod a+w -R ~/.eric6
